os 模块

常用函数

  • 判断文件是否存在
    os.path.exists

  • 获取当前目录
    os.getcwd()

  • 打印目录中的所有文件及目录
    os.listdir(os.getcwd())

  • 更改所在目录
    os.chdir('./path')

  • 合并路径
    os.path.join(os.getcwd(),os.listdir(os.getcwd())[0])
